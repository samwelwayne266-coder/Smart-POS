# 🛡️ Wayne Core | Smart POS System
## 🚀 Project Overview
Smart POS is a futuristic, high-performance retail suite. It uses a NoSQL backend to ensure that your inventory and sales data are always in sync.

## ✨ Elite Features
⚡ Quantum Pulse Scanner: Optimized camera logic with a 1.5s cooldown to eliminate duplicate scans.

💎 Glassmorphism UI: A high-tech aesthetic using Tailwind CSS for a deep-space, transparent look.

🔄 Real-time Core: Powered by Firebase, your Inventory and Sales sync across the globe in <100ms.

🔒 Secure Gateway: Integrated Google Authentication to keep the Manager and POS locked down.

## 🛰️ System Modules
### 1. Smart POS Terminal (pos.html)
** Quantum Scanner **: Features a 1.5s cooldown to prevent duplicate scans.

** Tactile Controls **: Users can use [+] and [-] to adjust quantities.

** Security **: Locked behind Google OAuth 2.0 for restricted access.

### 2. Inventory Manager (manager.html)
** Live Uplink **: Allows you to PUSH new data to the Firebase cloud.

** Stock Alerts **: Displays Red Alert styling when qty drops below 10.
